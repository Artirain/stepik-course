# Pro и Max — сравнение цен

!!! info "Что ты узнаешь"
    - Какие планы есть у Claude на 1 марта 2026
    - Как выбирать план под реальную нагрузку
    - Где проверять актуальные лимиты перед оплатой

## Введение

Планы и лимиты меняются, поэтому в этом уроке главное не «запомнить цифры», а понять логику выбора.

По состоянию на **1 марта 2026** у Anthropic есть планы **Pro**, **Max ($100)** и **Max ($200)**, где Max даёт существенно больше квоты использования, чем Pro.

!!! warning "Важно"
    Перед покупкой всегда сверяй условия в официальном источнике:
    [Claude plan tiers](https://support.claude.com/en/articles/9267304-about-claude-s-plan-tiers).

## Что важно сравнивать

| Параметр | На что смотреть |
|----------|-----------------|
| Цена | Фиксированная ежемесячная |
| Квота использования | Насколько хватает под твой дневной ритм |
| Доступ к возможностям | Нужен ли тебе Claude Code Web/интеграции/командная работа |
| Предсказуемость расходов | Подписка vs API-платёж по токенам |

## Быстрый ориентир по выбору

### Pro — старт и умеренное использование

Подходит, если:

- только входишь в Claude Code
- работаешь нерегулярно
- хочешь понятный фиксированный бюджет

### Max $100 — ежедневная разработка

Подходит, если:

- работаешь в Claude Code каждый день
- регулярно используешь большие задачи
- на Pro часто упираешься в лимиты

### Max $200 — высокая интенсивность

Подходит, если:

- Claude Code — основной рабочий инструмент
- много длинных и тяжёлых сессий
- важен максимальный запас квоты

## Когда смотреть в сторону API

Если тебе нужна автоматизация (CI/CD, массовые headless-запуски, программный контроль бюджета), часто лучше API-модель оплаты. Детали — в [Уроке 8](08-pro-plan-vs-api.md).

## Практика

1. Зафиксируй свой паттерн нагрузки: сколько часов в день ты реально работаешь с Claude Code.
2. Проверь официальный FAQ по планам и текущие ограничения.
3. Выбери минимальный план, который покрывает твою текущую нагрузку + небольшой запас.

## Итоги

- Выбор плана делается по нагрузке, а не «по максимуму на всякий случай»
- Pro — старт и умеренная интенсивность
- Max — для устойчивой ежедневной/интенсивной работы
- Официальные цены и лимиты всегда проверяй перед оплатой
