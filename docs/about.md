# О курсе

**«Полное руководство по Claude Code — вайбкодинг с нуля»** — это практический курс для разработчиков, которые хотят освоить Claude Code как основной инструмент для работы с кодом.

## Для кого этот курс

- Разработчики, которые хотят использовать ИИ-ассистенты в повседневной работе
- Те, кто уже пробовал Cursor или Copilot и хочет разобраться в Claude Code
- Опытные пользователи, которые хотят углубить знания: hooks, MCP, skills

## Формат

- **44 урока** в текстовом формате с примерами кода
- **Интерактивные квизы** для проверки знаний после каждого урока
- **Два раздела**: от основ до продвинутых возможностей
