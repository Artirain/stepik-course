# О курсе

**«Полное руководство по Claude Code — вайбкодинг с нуля»** — практический курс для разработчиков, которые хотят освоить Claude Code как рабочий инструмент.

## Для кого этот курс

- разработчики, внедряющие ИИ-ассистентов в ежедневную работу
- пользователи Cursor/Copilot/Codex, которые хотят глубже понять Claude Code
- опытные инженеры, которым важны hooks, MCP, skills и автоматизация

## Формат

- **44 урока** в текстовом формате с примерами
- **интерактивные квизы** после уроков
- **2 раздела**: от базовой установки до продвинутых сценариев
